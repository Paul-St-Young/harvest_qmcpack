qharv.reel.scalar_dat.single_column(df, column, nequil)[source]¶

calculate mean and error of a column

nequil blocks of data are thrown out; autocorrelation time is taken into account when calculating error The equilibrated data is assumed to have Gaussian distribution. Error is calculated for one standard deviation (1-sigma error).

Parameters:
  • df (pd.DataFrame) – table of data (e.g. from scalar_dat.parse)
  • column (str) – name of column
  • nequil (int) – number of equilibration blocks
Returns:

(ymean,yerr), where ymean is the mean of column, while yerr is the 1-sigma error of column
Return type:

(float,float)

qharv.reel.stat_h5 module¶

qharv.reel.stat_h5.absolute_magnetization(stat_fname, nequil, obs_name='SpinDensity', up_name='u', dn_name='d')[source]¶
calculate up-down spin density and the integral of its absolute value
first check that /SpinDensity/u and /SpinDensity/d both exist in the .stat.h5 file, then extract both densities, subtract and integrate
Parameters:
  • stat_fname (str) – name of the .stat.h5 file
  • nequil (int) – number of equilibration blocks to throw out
  • obs_name (str) – name of spindensity observable, default=SpinDensity
  • up_name (str) – default=u
  • dn_name (str) – default=d
Returns:

(rho,rhoe,mabs_mean,mabs_error), rho and rhoe have shape (ngrid,), where ngrid is the total number of real-space grid points i.e. for grid=(nx,ny,nz), ngrid=nx*ny*nz. mabs = int |rho_up-rho_dn|
Return type:

(np.array,np.array,float,float)

qharv.reel.stat_h5.mean_and_err(stat_fname, obs_path, nequil)[source]¶

calculate mean and variance of an observable from QMCPACK stat.h5 file

assume autocorrelation = 1.

Parameters:
  • stat_fname (str) – .stat.h5 filename
  • obs_path (str) – path to observable, e.g. ‘gofr_e_1_1’
  • nequil (int) – number of equilibration blocks to throw out
Returns:

(val_mean,val_err), the mean and error of the observable, assuming no autocorrelation. For correlated data, error is underestimated by a factor of sqrt(autocorrelation).
Return type:

(np.array,np.array)
